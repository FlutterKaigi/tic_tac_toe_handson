## ハンズオン概要

今回はのテーマは **Riverpod** です。

Flutter 開発における状態管理の手法として注目を集めている Riverpod の基本的な使い方を、三目並べアプリの作成を通して一緒に学んでいきましょう。後半のチャレンジ企画では、 Firebase のFirestore を使ったリアルタイム対戦機能の実装を予定しています。
